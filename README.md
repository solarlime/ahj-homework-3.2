# Simple task manager

### Features:
- Add / remove your tasks
- Pin important tasks to top & unpin them
- Delete old tasks
- Search your tasks

All information stores in your localStorage.

Try it on [GitHub Pages](https://solarlime.github.io/ahj-homework-3.2/)!
